
# 📱 Desafio QR Code - Frontend Mentor

Este projeto é um desafio prático de front-end inspirado no site [Frontend Mentor](https://www.frontendmentor.io/). O objetivo é criar um componente de cartão com um QR Code, seguindo boas práticas de HTML e CSS responsivo.

## 🧾 Descrição

A página apresenta um **cartão centralizado** contendo:
- Uma imagem de QR Code.
- Um título com chamada à ação.
- Uma descrição explicativa.
- Estilização responsiva e uso de fontes do Google.

## 📁 Estrutura do Projeto

```
📦 qr-code-project
├── index.html
├── css/
│   ├── reset.css
│   ├── style.css
│   └── responsive.css
├── image-qr-code.png
├── favicon-32x32.png
```

## 🚀 Funcionalidades

- ✅ Layout centralizado e adaptável a telas menores.
- ✅ Tipografia moderna usando a fonte **Outfit** do Google Fonts.
- ✅ Imagem de QR Code com texto explicativo.
- ✅ Estrutura de HTML semântico e limpa.

## 🛠 Tecnologias Utilizadas

- **HTML5**
- **CSS3** (com separação de estilos por responsividade e reset)
- **Google Fonts** (`Outfit`)
